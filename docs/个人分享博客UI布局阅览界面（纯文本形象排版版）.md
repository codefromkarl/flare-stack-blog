## 四、关键排版补充说明

- 所有间隔符「──────────」均对应UI中的“轻量分隔线”，厚度1px，颜色浅灰，无生硬感；

- 表格仅用于呈现导航栏栏目分布，对应实际UI中的横向均匀排列，无边框、无背景；

- 文字层级：标题（轻量加粗+低饱和深色）→ 正文（浅灰色）→ 辅助文字（浅灰色+小字），层级清晰不突兀；

- 留白比例均为相对值，实际可根据你现有博客尺寸微调，核心是保持柔和不拥挤。
